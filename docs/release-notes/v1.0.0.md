## 系统要求

- macOS **14.0 (Sonoma)** 或更新
- Apple Silicon (M1 / M2 / M3 / M4) 或 Intel Mac (x86_64) — **universal binary**，同一个 DMG 自动适配

## 主要功能

### 浏览
- **侧边栏树形文件夹** — 拖文件夹到侧边栏自动加入；自动构建子文件夹树 + 图片数 badge
- **缩略图网格** — Toolbar 滑块调缩略图尺寸（80~280pt），UserDefaults 持久化
- **6 种排序** — 文件名 / 修改时间 / 创建时间 / 大小 / 类型 / 路径，正反序

### 看图
- **单击进内嵌预览** — 文件名跟随 toolbar，方向键 ←→ 切换零延迟（±2 张预加载）
- **双击进沉浸式 QuickViewer** — 全窗口查看，自适应 fit / 1:1 切换
- **缩放与拖拽** — 滚轮 / 双击切换 / 鼠标拖动平移
- **键盘快捷键** — Space 进入查看器、ESC 退出、方向键导航、`F` 切换全屏

### Inspector（⌘I 切换）
- EXIF 元信息：相机型号 / 镜头 / 光圈 / 快门 / 感光度 / 焦距
- 拍摄时间 / 文件大小 / 像素尺寸
- GPS（如有）

### 外观
- **跟随系统 / 强制深色 / 强制浅色** 三档，UserDefaults 持久化
- macOS 26 风格毛玻璃 Toolbar、青紫色光晕
- 中英双语显示名（英文系统显示 "Glance" / 中文系统显示「一眼」）

## 数据安全 & 隐私

- ✅ App Sandbox 沙盒（用户授权才能读特定文件夹）
- ✅ Apple Developer ID 签名 + 公证（Hardened Runtime）
- ✅ Security Scoped Bookmark — 重启后自动恢复授权
- ❌ **零网络请求 / 零数据上传 / 零遥测**

## 已知 limitations

- 不支持 RAW / HEIC / GIF 动画（V1 仅 JPG / PNG / TIFF / BMP）
- 离线首次启动 Gatekeeper 验证可能短暂等待（DMG 公证票已 staple，.app 单独 staple 在 v1.0.1 加）
